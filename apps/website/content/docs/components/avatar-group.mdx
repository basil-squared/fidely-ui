## Usage

```tsx
import { Avatar, AvatarGroup } from '@fidely-ui/react/avatar'
```

```tsx
<AvatarGroup>
  <Avatar.Root>
    <Avatar.Fallback />
    <Avatar.Image />
  </Avatar.Root>
</AvatarGroup>
```

To reduce the overlapping on each you can pass the **spaceX** to the AvatarGroup component.

<ExampleTabs name="avatar-group-space" scope="avatar" />

To remove both the overlapping and attach pass **`attach={false}`** and **`overlap={false}`** to the AvatarGroup component.
