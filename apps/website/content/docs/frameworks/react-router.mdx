## Get started with our template

The easiest way to get started is to use our **React Router v7 + TypeScript** template.  
It comes preconfigured with Panda CSS, Ark UI, Fidely UI v2 and theme support.

```bash
npx degit https://github.com/fidely-ui/react-router-template
```
